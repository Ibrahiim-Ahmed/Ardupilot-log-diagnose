## AI-Assisted Flight Log Diagnosis (Prototype)

This project is an initial prototype for automated diagnosis of flight issues
from ArduPilot logs.

Current features:
- Simulated flight log generation
- Feature extraction (vibration, voltage)
- Rule-based root cause detection
- Explainable diagnostic output

Future work:
- Integration with real ArduPilot logs using pymavlink
- Machine learning models for classification
- Root-cause retrieval system
